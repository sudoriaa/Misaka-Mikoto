# 御坂美琴应援站（Misaka-Mikoto.com）  
​**⚡ 基于 Vue 3 + SCSS + Iconify 开发的超电磁炮应援站**​  

[网站预览]

---

## ✨ 项目介绍  
专为《某科学的超电磁炮》人气角色御坂美琴打造的粉丝应援平台，包含以下核心功能：  
- 角色资料卡（能力值/经典场景）  
- 动态台词轮播系统  
- 高清图片墙（懒加载优化）  
- 粉丝互动留言板  

​**技术栈**​  
| 类别 | 技术选型 |  
|------|----------|  
| 框架 | Vue 3 (Composition API) |  
| 样式 | SCSS + CSS变量 |  
| 图标 | Iconify |  
| 构建 | Vite |  
| 部署 | Vercel Edge |
