​**✨ 项目介绍**​
这是一个为《魔法禁书目录》/《某科学的超电磁炮》中的角色 ​御坂美琴（Misaka Mikoto）​​ 开发的粉丝应援网站，包含角色介绍、经典台词、图片墙、应援留言板等功能。

​技术栈：​​

​前端框架: Vue 3 (Composition API)
​样式预处理器: SCSS
​图标库: Iconify
​构建工具: Vite
​部署: GitHub Pages / Vercel

​🎨 设计风格​
​主色调: #9e2f8c（美琴的代表色）
